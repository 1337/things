What is browser.php?

Sparta File Manager 3 (codename browser.php) is a file manager in a single PHP file. Its main features are:

  • Single PHP file structure
    Easy to install, configure and upgrade.
  • Code in it
    browser.php has a built-in code editor, which supports syntax highlighting (using EditArea).
  • Download any file
    browser.php has a built-in file download feature that allows you to download any accessible file from the server. 
  • File management
    Ability to batch-delete files; modify files and folders, such as create, change permissions and delete
  • View-only mode
    Also built-in, this mode allows users to download any file from the directory, but cannot modify its contents.

Requirements

Web server, with PHP version >= 4.03

20kB free space

Installation

  1. Go to http://www.friendpaste.com/6IGFrgCYDykIzxbKAjkFib and copy the code to a new text file.
  2. Save as 'browser.php'.
  3. Upload the file to the server.

Warnings

Without a password or authentication set for this browser, it can be exploited by hackers as a backdoor. Beware!
