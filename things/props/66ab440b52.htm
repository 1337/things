• Use meaningful markup.
    For example, don't use <h3> for formatting purposes, or <p><b></b></p> for headings.
  • Use minimal markup.
    When <p> makes sense, don't do <div class="text">.
  • Use classes and ids correctly.
    class for multiple occurrences.
    ids for single occurrences (errors occur if multiple objects are assigned the same id)
    names for backwards compatibility and forms.
  • Know your tags.
    There are many tags that will help accessibility and search engine optimisation. Use them where possible. Try these ones for kicks:
    • <abbr title="abbreviation">abbr</abbr>
    • <acronym title="search engine optimization">seo</acronym>
    • <address>1 Infinite Loop</address>
    • <caption>Table 3.2.  Raw Sales Data</caption> (same idea as table captions in word, I suppose.)
    • <code><?php echo “this is a code example”;?></code>
    • <em>Emphasis</em>
    • <strong>Even stronger emphasis</strong>
    • <cite>Doe, J. (). Blah blah. 2nd Ed ed. : Publisher.</cite>
    • <dfn title="Definition">dfn</dfn>
    • <dl> definition list, <dt> definition text, <dd> definition definition
      <dl>
          <dt>term 1</dt>
          <dd>definition of the first term.</dd>
      <dl>
    • <del>crossed out</del>
    • <ins>inserted text (underlined)</ins>
    • <label for="textbox"><input type='text' id='textbox' /></label>
    • <samp>OUTPUT FROM SCRIPTS<samp>
    • User, please type this in the command line: <kbd>rm -rf /</kbd>
    • <th scope="row"> (this table header represents a row in the table) and
      <th scope="col"> (this table header represents a column in the table) 
    • <var>variable_x</var>
  • Know your CSS properties.
    • font-stretch: no browsers support it.
    • font-weight: bolder (bolder than the INHERITED value)
    • font-style: oblique (slanted) or italic (specific variant of the font)
  • Menus should have smaller font size than the body text.
    If the content is more important, then the content should be of larger font.
  • Exploit colours and their meanings.
    Beware of cultural differences that might cause a change in these colour meanings. 
    • Red: Power, energy, love, passion, aggression, danger
    • Blue: Trust, conservative, secure, clean, sorrowful, order
    • Brown: Luxurious, reliability, comfort, endurance, earth
    • Green: Nature, earth, health, jealousy, renewal.
    • Orange: Fun, happiness
    • Yellow: Optimism, hope, philosophy, cowardice 
    • Purple: Royalty, mystery, religion 
    • Gray:Modesty, sadness, decay, elegance 
    • Silver:Intellect, futurism 
    • Black:Power, sexuality, sophistication, mystery, fear, unhappiness 
    • White:Purity, cleanliness, precision, innocence, sterility, death
  • Contrast and shadows.
    Objects with more contrast appear thicker.
    Objects with less contrast appear thinner. 
  • Utilise layouts.
    Absolute column positioning is a tried-and-true method for making website columns.
  • Negative margins.
    consider this one. the image centers dead on middle:
    position: absolute;
    left: 50%;
    width: 1000px;
    margin: -500px;
  • Use em-sized fonts (modular scale), and the 960 grid system.
    16px and 24px are the two golden pixel sizes. That corresponds to
    1em and 1.5em.
